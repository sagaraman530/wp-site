Supported Charge Station Vendors and Models

ChargePanel’s innovative platform for electric vehicle charging is compatible with leading hardware manufacturers. The ChargePanel platform is hardware-agnostic and supports EV chargers using OCPP (JSON), version 1.5 or higher. ChargePanel’s EV charging software is compatible with vendors from around the globe, with over 50 charge station models already connected and new charging stations being added all the time.

OCPP

Open Charge Point Protocol

The goal for Open Charge Point Protocol (OCPP) is to offer a uniform solution for the method of communication between charge point and central system. With this protocol it is possible to connect any central system with any charge point, regardless of the vendor.

OCPP, which stands for Open Charge Point Protocol, is a communication protocol used within the electric vehicle (EV) charging infrastructure. It has been developed to enable standardized communication between charging stations and central management systems (or charging station management systems), making it possible to easily manage and monitor the charging process. Installers of charging stations benefit from OCPP by being able to connect and integrate different charging stations from different manufacturers in the same network.

The protocol defines a common set of commands and messages that allow communication between stations and central management systems, regardless of the hardware used. This makes it easier to install and maintain charging infrastructure, as you are not tied to a specific manufacturer’s products. Charging station hardware manufacturers benefit from OCPP by being able to create products that can communicate with different central management systems on the market. By implementing OCPP in their charging stations, manufacturers can offer interoperability and compatibility with different networks and operators. This enables them to reach a wider market and increases flexibility for their customers. There are several versions of OCPP, with each version offering different features and enhancements. The most common versions are OCPP 1.5, OCPP 1.6 and OCPP 2.0.

OCPP 1.5 is the older version and supports basic charge control and monitoring functions. OCPP 1.6 is an intermediate version that introduces more features that enable more advanced charge management and integration with external systems. OCPP 2.0 is the latest version and has further improvements and expanded features to meet the growing needs of the charging infrastructure, including added smart charging functionalities, improved transaction handling, device management, support for ISO 15118, and more.
